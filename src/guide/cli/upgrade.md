# Upgrade

Download and install the latest version of DFtpS.

```sh
dftps upgrade
```

::: tip Auto-detection
The upgrade command automatically detects your OS and architecture to download the correct binary.
:::

## Supported platforms

| Platform | Binary |
|----------|--------|
| Linux x64 | `dftps-linux-x64` |
| macOS Intel | `dftps-macos-x64` |
| macOS Apple Silicon | `dftps-macos-arm64` |
| Windows x64 | `dftps-windows-x64.exe` |

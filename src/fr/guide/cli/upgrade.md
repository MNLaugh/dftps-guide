# Mise à jour

Télécharge et installe la dernière version de DFtpS.

```sh
dftps upgrade
```

::: tip Détection automatique
La commande upgrade détecte automatiquement votre OS et architecture pour télécharger le bon binaire.
:::

## Plateformes supportées

| Plateforme | Binaire |
|------------|---------|
| Linux x64 | `dftps-linux-x64` |
| macOS Intel | `dftps-macos-x64` |
| macOS Apple Silicon | `dftps-macos-arm64` |
| Windows x64 | `dftps-windows-x64.exe` |
